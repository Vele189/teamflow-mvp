# TeamFlow MVP

TeamFlow MVP is a monorepo that contains the frontend, backend, and shared packages for the TeamFlow project.

## Linux Development Environment

Ubuntu 24.04 LTS is the recommended Linux development environment.

To set up your machine:

```bash
chmod +x scripts/setup.sh
./scripts/setup.sh
```

For detailed setup instructions, see:

```
docs/linux-development-setup.md
```
